About Me

I’m Arya Rizal, a Mobile Engineer & Fullstack Developer with over 3 years in mobile development, 5 years in web development, and 2 years in backend engineering. I specialize in building high-quality, scalable applications using Clean Architecture (MVVM-based Modular Architecture) with a strong emphasis on performance optimization.

I have extensive experience developing mobile applications using Flutter, designed with an offline-first approach powered by SQLite, managing over 25 data tables with efficient query handling and synchronization systems. My apps are adaptive for both iOS and Android, seamlessly integrated with Firebase Cloud Messaging (FCM) and modularized using Melos for scalable code management.

Beyond mobile, I also build clean and responsive web applications and design backend systems using Node.js with proper architecture, deployed on Google Cloud Platform (GCP) with Kubernetes (GKE) for reliability and scalability.

For me, great software is more than just functional it must be well-structured, maintainable, and deliver a seamless user experience across every platform.
